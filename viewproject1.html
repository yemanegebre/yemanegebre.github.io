Project

Ecommerce Website using Node.js and React Redux

As a full-stack developer at Adeymeselesh Software Development PLC, I designed and developed an ecommerce website using Node.js and React Redux. This ecommerce website allows customers to purchase products using PayPal and credit card online payment methods.
Some of the key features of the ecommerce website include:

• Responsive design that allows customers to shop from any device
• Efficient and secure checkout process with options for PayPal and credit card payments
• Integration with a third-party shipping service to ensure timely delivery of products
• Administration dashboard that allows the business owner to manage products, orders, and customer information
I developed this ecommerce website using a range of modern web development technologies, including Node.js, React Redux, and Bootstrap. The site is hosted on a cloud platform, and I implemented robust security measures to protect customer data and prevent fraud.

The ecommerce website has been a tremendous success for the business, helping to increase sales and reach new customers. I am proud of my work on this project and am excited to continue developing innovative solutions for businesses and organizations.

You can view the website at https://web-shop.adeymeselesh.de/.
